#  Interkacija Čovek Računar 2025

Izvorni kod aplikacije sa vežbi iz predmeta Interkacija Čovek Računar na 4 godini Tehničkog Fakulteta Univerziteta Singidunum.

Aplikacija je razvijena korsteći sledeće tehnologije:
- Angular 20
- Boostrap 5.3
- Fonts Awesome 7
- Axios HTTP Client
- Rasa Open Source

## Pokretanje Projekta

Kako bi pokrenuli Angular projekat potrebno je da posedujete Node 22 i Bun runtime na vašem računaru.
Aplikaciju pokrećete uz pomoć komande `bun start`
